MATH CALCULATOR

Example: calculate the expression: 243.4+345*98.1

  1. Enter the number 345 using the online numeric key pad. In case you made a mistake, use the backspace button ⇦ or clear the entire input box using "CLR" button
  2. Click on "*" button, then enter the number 98.1 and press the "=" button to get the intermediate result
  3. Click the "+" button, enter 243.4 and press "=" button to get the final result shown in lower box
  4. Use the "⇧" and "⇩" keys to move the numbers to/from the memory register (lower box)
